# Global Tags

This section is optional. You may list multiple `tags` as key/value pair. The tags will be added to any provisioned resources.

example:

```json
{

  "global_tags": {
    "Team": "awesome",
    "Environment": "devel",
    "Department": "product",
    "Product": "deployer",
    "DeployedBy": "jsmith"
  }

}
```
